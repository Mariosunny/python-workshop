What is Python?

Python is an interpreted, object-oriented, high-level programming language with highly dynamic semantics. It is used by millions of programmers worldwide to create enterprise-level applications and simple scripts.

Notable Features of Python:

  • Easy to read syntax
  • Simple debugging process
  • Built-in data structures and dynamic typing and binding
  • Wide selection libraries from image processing to web scraping
  • Large and active community of developers

Comparison to other Programming Languages

Python has several major differences to traditional languages.

  • Types do not need to be specified when declaring variables. a = 5 is valid syntax, int a = 5 is not valid syntax.
  • Identation levels are meaningful. In C++/Java, blocks of code are defined by curly braces {}, whereas in Python identation levels define a block of code.
  • Semicolons ; are optional.
  • Python is explicit rather than implicit. For example, numbers must be explicitly coverted to strings before they can be concatenated with another string.
  • Everything is an object, including numbers, strings, lists, functions, and classes.

Java code

public class Main {
   public static void main(String[] args) {
      System.out.println(fib(6));
   }
   public static int fib(int n) {
      if(n <= 1) {
         return 1;
      }
      return fib(n - 2) + fib(n - 1);
   }
}

Check it out: try typing in import this into your Python interpreter.

Python code

def fib(n):
   if n <= 1:
      return 1
   return fib(n - 2) + fib(n - 1)
print(fib(6))

Variable Assignment and Numbers

To declare a variable and assign it a value, simply type the name of the variable, then the variable assignment operator =, and then the value of the variable.

x = 5

Python supports the basic number types: int, for integers, and float, for real numbers.

To print an expression to the console, use the built-in print() function.

x = 5.4
print(x) # prints 5.4 to the console

Python supports all of the basic math operations.

a = 5 + 5 # addition
b = 10 - 5 # subtraction
c = 3 * 9 # multiplication
d = 5 / 8 # division
e = 5 % 2 # modulo
f = 2 ** 3 # exponetiation
g = 8 // 3 # integer division

User Input

User input is collected via the built-in input() function.

num = input("Enter a number: ")

All input is read as a string. To convert to a number, use int() or float().

a = int(num)
b = float(num)

Loops

Python supports both for loops and while loops

Looping through the numbers 0 to 9: 

for i in range(10):
   print(i)

Using a while loop:

i = 0
while True:
   print(i)
   i += 1

Looping through a list:

animals = ['cow', 'pig', 'duck', 'cat']
for animal in animals:
   print(animal)

Looping through a list with enumerate():

animals = ['cow', 'pig', 'duck', 'cat']
for i, animal in enumerate(animals):
   print(i, "\t", animal)

If Statements

Python supports the boolean types True and False.

a = True
b = False

Python also supports if/elif/else statements.

a = 9
if a > 9:
   print("greater than 9")
elif a > 5:
   print("greater than 5")
else:
   print("less then or equal to 5")

Python also supports the three basic boolean operators, and, or, and not.

a = 9
b = 14

c = a >= 9 and b > 12 #True
d = a >= 9 or b > 12 #True
e = not a == 9 #False

Functions

To declare a function, use the keyword def. Like any other objects in Python, with functions the type is not declared.

def func1(a, b):
   if a > 3:
      return a**2 + b/3
   else:
      return (a - 5)/b

Modules

A module is simply a Python file. Modules are the basic unit of code organization in Python projects.

Like most languages, Python includes a standard library which contains many useful APIs which can be imported as modules.

One of the most commonly used modules is the math module. To import a module, use the import statement followed by the module name.

import math

print(math.sqrt(10)) #square root
print(math.floor(10.384)) #floor
print(math.log(300, 12)) #log base 12
print(math.pi) #Pi constant: 3.141592...
